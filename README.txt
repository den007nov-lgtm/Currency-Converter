# Currency-Converter

Автор: Денис Новокщенов / sgiman

Год создания:2026

API ПриватБанка является открытым и доступным по следующим URL:

Тип курса	URL	Описание

Наличные	https://api.privatbank.ua/p24api/pubinfo?json&exchange&coursid=5	Курсы для наличных операций

Безналичные (карты)	https://api.privatbank.ua/p24api/pubinfo?exchange&coursid=11	Курсы для безналичных расчётов

Данное приложение представляет собой конвертер валют

Который получает актуальные курсы валют из API ПриватБанка 

(PrivatBank API) и выполняет конвертацию из USD в UAH (доллар США в украинскую гривну).
