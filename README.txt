#  Training Planner - Планировщик тренировок

**Автор:** Денис Новокщенов / sgiman, 2026

Приложение для планирования и учёта тренировок с возможностью фильтрации, редактирования и экспорта данных.

---

##  Возможности

-  Добавление тренировок с указанием даты, типа и длительности
-  Строгая валидация ввода (формат даты YYYY-MM-DD, положительная длительность)
-  Фильтрация тренировок по диапазону дат и типу
-  Редактирование и удаление тренировок
-  Автоматическое сохранение в JSON-файл
-  Экспорт данных в CSV
-  Подсчёт статистики (общее количество и длительность тренировок)
-  Удобный табличный интерфейс
